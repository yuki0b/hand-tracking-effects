# Hand Tracking Effects

Python + MediaPipe 기반 손동작 이펙트 프로젝트

---

## Effects

-  Fire Effect
-  Heart Effect
-  Thanos Gauntlet
-  Rason Effect

---

##  Hand Gestures

| Gesture | Effect |
|---|---|
|  검지 하나 | Fire |
|  엄지 + 검지 | Heart |
|  주먹 | Thanos |
|  검지 + 중지 십자 | Rason |

---

- Python 3.12
- OpenCV
- MediaPipe
